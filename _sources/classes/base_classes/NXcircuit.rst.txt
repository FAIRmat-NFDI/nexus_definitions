**Status**:

**Description**:

  .. collapse:: Base class for documenting circuit devices. ...

      Base class for documenting circuit devices.

      Electronic circuits are hardware components that connect several electronic components to achieve
      specific functionality, e.g. amplifying a voltage or convert a voltage to binary numbers, etc.

**Symbols**:

  .. collapse:: Constant to be used in the definition: the number of channels of the ...

      Constant to be used in the definition: the number of channels of the
      circuit board.

  **N_channel**: Number of channels of the circuit board.
