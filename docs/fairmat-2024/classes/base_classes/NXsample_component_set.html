3.3.1.150. NXsample_component_set¶

Status:

base class, extends NXobject

Description:

Set of sample components and their configuration. ...

Set of sample components and their configuration.

The idea here is to have a united place for all materials descriptors that are not part of the individual sample components, but rather their configuration.

Symbols:

n_comp: number of components

Groups cited:

NXsample_component_set, NXsample_component

Structure:

@components: (optional) NX_CHAR

Array of strings referring to the names of the NXsample_components. ...

Array of strings referring to the names of the NXsample_components. The order of these components serves as an index (starting at 1).

concentration: (optional) NX_FLOAT (Rank: 1, Dimensions: [n_comp]) {units=NX_ANY}

Concentration of each component

volume_fraction: (optional) NX_FLOAT (Rank: 1, Dimensions: [n_comp]) {units=NX_ANY}

Volume fraction of each component

scattering_length_density: (optional) NX_FLOAT (Rank: 1, Dimensions: [n_comp]) {units=NX_ANY}

Scattering length density of each component

SAMPLE_COMPONENT: (optional) NXsample_component

Each component set can contain multiple components.

SAMPLE_COMPONENT_SET: (optional) NXsample_component_set

For description of a sub-component set. Can contain multiple components itself.
