3.3.1.15. NXapm_volt_and_bowl¶

Status:

base class, extends NXprocess

Description:

Base class for the configuration and results from a voltage-and-bowl ToF correct ...

Base class for the configuration and results from a voltage-and-bowl ToF correction algorithm.

The voltage-and-bowl correction is a ata post-processing step to correct for ion impact position flight path differences, detector biases, and nonlinearities.

Symbols:

The symbols used in the schema to specify e.g. dimensions of arrays.

n: Number of ions spatially filtered from results of the hit_finding algorithm NXapm_hit_finding from which an instance of a reconstructed volume has been generated. These ions get new identifier assigned in the process - the so-called evaporation_identifier, which must not be confused with the pulse_identifier!

Groups cited:

NXprogram, NXserialized

Structure:

raw_tof: (optional) NX_FLOAT (Rank: 1, Dimensions: [n]) {units=NX_TIME}

Raw time-of-flight data without corrections.

calibrated_tof: (optional) NX_FLOAT (Rank: 1, Dimensions: [n]) {units=NX_TIME}

Calibrated time-of-flight.

PROGRAM: (optional) NXprogram

SERIALIZED: (optional) NXserialized
