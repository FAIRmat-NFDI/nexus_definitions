3.3.1.131. NXphysical_process¶

Status:

base class, extends NXobject

Description:

A planned or unplanned process which results in physical changes in a specified ...

A planned or unplanned process which results in physical changes in a specified material.

A physical change involve changes only in intermolecular forces, not in the chemical bonds. Examples include sample preparation, material transformation, or (partially) destructive measurements.

Symbols:

No symbol table

Groups cited:

NXnote

Structure:

start_time: (optional) NX_DATE_TIME

ISO 8601 formatted time code (with local time zone offset to UTC information ...

ISO 8601 formatted time code (with local time zone offset to UTC information included) when this process started.

end_time: (optional) NX_DATE_TIME

ISO 8601 formatted time code (with local time zone offset to UTC information ...

ISO 8601 formatted time code (with local time zone offset to UTC information included) when this process ended.

description: (optional) NX_CHAR

Short description of the activity.

method: (optional) NX_CHAR

Method by which this process was performed.

notes: (optional) NXnote

This can be any data or other descriptor acquired during the physical process ...

This can be any data or other descriptor acquired during the physical process (NXnote allows to add pictures, audio, movies). Alternatively, a reference to the location or a unique identifier or other metadata file. In the case these are not available, free-text description.
