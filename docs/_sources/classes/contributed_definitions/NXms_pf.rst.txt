NXms_pf
=======

**Status**:

  base class, extends :ref:`NXprocess`

**Description**:

  Base class to store a pole figure (PF) computation.

**Symbols**:


  **n_y**: Number of pixel per pole figure in the slow direction.

  **n_x**: Number of pixel per pole figure in the fast direction.
