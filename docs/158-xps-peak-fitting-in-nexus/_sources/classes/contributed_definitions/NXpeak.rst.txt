**Status**:

**Description**:

  .. collapse:: Base class for describing a spectral peak, its functional form, and support valu ...

      Base class for describing a spectral peak, its functional form, and support values
      (i.e., the discretization (points) at which the function has been evaluated).

  **n_values**: Number of discretization points at which the functional form of the peak has been evaluated.
