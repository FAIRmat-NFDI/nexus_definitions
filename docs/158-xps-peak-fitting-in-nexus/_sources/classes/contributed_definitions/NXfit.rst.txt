=====
NXfit
=====

**Status**:

  base class, extends :ref:`NXobject`

**Description**:

  Description of a fit procedure.

**Symbols**:

  The symbols used in the schema to specify e.g. dimensions of arrays.

  **n_points**: Number of discretization points inside the fitting region.

  **n_dependent**: Number of dependent variables to be fitted (for multidimensional/multivariate fit.)
